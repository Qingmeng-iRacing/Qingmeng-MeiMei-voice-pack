--- ⚙️ 安装指南 (Installation) ---

（该步骤图为目录中“ini文件必看设置图片”）
首先打开此电脑（我的电脑），点击文档-iRacing-app.ini
打开您的 app.ini 文件。
找到 [Audio] 这个区块。
在 [Audio] 区块里，找到下面这一行：
overrideSpccRate=0
将 0 修改为 1：
overrideSpccRate=1
设置成功后按Ctrl+S保存文件，关闭。

然后：
1. 解压压缩包主体，得到名为「Qingmeng-MeiMei-Spotter-v1.1」的文件夹，然后打开游戏安装目录。方法为：打开Steam游戏库，找到iRacing图标右键-管理-浏览本地文件（官网用户直接右键桌面iRacing UI快捷方式-打开文件所在位置-回退到iRacing目录）
2. 将该文件夹完整复制到 iRacing 游戏安装目录下的 \ sound \ spcc，将语音包主体文件放入spcc文件夹内。
3. 启动游戏后，点击右上角设置【Settings】，点击声音设置页面 【Audio】，左侧选项栏点击【AI Spotter】，找到【AI Spotter Options】栏目。
在【Voice Pack】下菜单中选择【Qingmeng-MeiMei-Spotter-v1.1】，点击页面右下角【Apply&Close】后，重启游戏后即成功安装。

墙裂推荐的【AI Spotter】设置：

👉关注【AI Spotter Options】栏目

👉将【Chattiness】下拉菜单中选择【High】★★★★★（非常重要）

👉启动【Report laps】功能✔

👉启动【Pit Box Countdown】功能✔

👉将【Report Laps】下拉菜单中选择【Time】

选择推荐的【AI Spotte】设置：

👉关注【AI Spotter Controls】栏目

👉将【Spotter Damage Report】绑定快捷按键，即可实现（按下绑定按键）启动主动车损报告功能

👉将【Spotter Weather Report】绑定快捷按键，即可实现（按下绑定按键）启动主动天气报告功能

（您按照以上设置可彻底解锁该语音包全部能力）
